tutors-html

El paquete tutors-publish-html es una herramienta que permite la compilación a HTML de sitios web estáticos escritos en Markdown, y siguiendo una estructura definida en el proyecto tutors.

Instalación

Al momento de escritura de la documentación, la versión más reciente de tutors-publish-html es la 3.0.3. En la terminal del sistema ejecute el siguiente comando para obtener la versión 3.0.3 npm install --global tutors-publish-html@3.0.3, y si desea obtener la versión más reciente utilice el comando npm install --global tutors-publish-html.

Es posible que al actualizar Node.js tenga que volver a instalar tutors-publish-html.

Para llamar a este paquete de compilación de sitios, debe ejecutar el comando tutors-html en terminal.
